This project is the number-guessing game option for Penn State University Park's CMPSC132 final project.

High-level Description:

The game is terminal-based. There are three difficulty levels (Easy, Medium, Hard), which vary the range of possible numbers and the number of allotted attempts. The user receives feedback on each guess. The game supports replayability. 

How to play:

The user is first prompted for the difficulty level they want to play. The game then starts and prompts the user for a guess. For each guess, the user receives feedback on the accuracy of the guess. This loop continues until the user runs out of attempts or inputs a correct guess. At the conclusion of each game, the user is asked to replay.
